Quick and Dirty

open the website in incognito/private browsing mode

Permanent solution for Google Chrome

menu -> settings -> show advanced settings -> content settings -> cookies -> manage exceptions
then add each url and set to 'block'

Why?

These sites are using cookies to track client page views so by ignoring/rejecting the cookies, the site cannot keep track of the number of articles read.

Tested on 

  • theglobeandmail.com
  • thestar.com
  • nationalpost.com
